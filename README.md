conway game of life
# Название проекта : conway game of life (игра жизнь)

## Краткое описание и цели 

Цель задания: Разработать приложение, демонстрирующее игру "Game of Life" и отображающее ее на экране с помощью графической библиотеки SDL2. Сборка проекта должна осуществляться через CMake.
Описание задачи: Напишите программу на языке C++, которая реализует клеточный автомат "Игра жизнь" (Game of Life) и отображает его состояние в виде графической сетки на экране.

## Требование к программе

 - Используйте язык программирования C++
 - Используйте библиотеку SDL2 для графического вывода
 - Добавьте возможность установки/удаления клеток с помощью мыши
 - Запуск игры должен происходить на чистом поле в режиме паузы для первоначальной расстановки клеток

## Компиляция и запуск

- Для компиляции запустите команду : cmake <путь>. Где путь ваша директория в которой находится проект

- Для запуска программы введите команду : make.

 ## Как запускать игру
 - Для работы программы нужно передать три аргумента коммандной строки： размеры поля и скорость. 
 - Пример ./main 200 200 10. 
 - Игра изначально в режиме паузы, выставите клетки нажатием правой или левой кнопки мыши, после того как выставили клетки, нажмите пробел и игра начнет работать
 - Чтобы поставить программу на паузу - нажмите пробел.
 - Чтобы ускорить программу - нажмите стрелку вверх.
 - Чтобы замедлить программу стрелку вниз.
 - Чтобы добавить или убрать новую клетку - нажмите левую кнопку мышки.
 - Чтобы отчистить экран, нажмите на клавишу C или c (английские буквы).
 - Чтобы перезапустить программу, нажмите на клавишу R или r.
